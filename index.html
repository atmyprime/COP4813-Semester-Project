This project is part of the CIS 4813 course requirements: real-life need, authentication, SQL storage, validation, and responsive design.

How It Works

  1. Create an account and set your calorie target.
  2. Choose your diet style (balanced, low carb, high protein, etc.).
  3. Pick how many days you want a plan for.
  4. SmartPlate generates meals using recipes stored in the database.

Your plans are saved so you can come back anytime and view them.
